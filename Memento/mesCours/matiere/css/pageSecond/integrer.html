Deux manières d'intégrer le css

Retour

  • Dans la feuille HTML:
    Il suffit dans les balises HEAD d'intégrer une balise "style" et d'y rentrer ses ligne CSS
  • Dans une feuille secondaire:
    Il faut créer un fichier style.css et mettre une balise link dans la feuille HTML dans le HEAD pour créer un lien entre les deux fichiers.
    Il ne nous restera plus qu'à rentrer le code CSS désirer dans la page secondaire, les modifications rentreront dans le résultat final
