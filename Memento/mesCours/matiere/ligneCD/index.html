La ligne de commande

Retour

ls

  • ls:Afficher la liste du dossier courant
  • ls -l:Lister (version longue)
  • ls -a:Affiche aussi les fichiers caché
  • ls -al:Affiche tout les fichiers, même cachés

Chemin/arborescence

  • pwd:Affiche le chemin du dossier courant.
  • cd /home/utilisateur/dossier :Se déplacer vers le chemin souhaité
  • c .. :Remonter au dossier parent

Les paquets

  • Sudo -s:Se mettre en super-utilisateur (à éviter)
  • Sudo apt + fonctionDesiree: Appliquer une fonction en tant que superutilisateur (à privilégié car plus prudent)
  • apt update: Sert à déterminer si une mise à jour est nécessaire parmis la liste des paquets
  • apt upgrade: Met à jour les paquets
  • apt search "caractère":Rechercher un paquet
  • sudo dpkg -i atom-amd4.deb:Installation de logiciel (ici par exemple => Atom, éditeur de texte)

Gestion de documents

Création de document

  • mkdir nom_du_repertoire:Création d'un répertoire dans le dossier courant
  • mv nom1 nom2:Renommer le nom1 en nom2
  • mv nom ~/Documents:déplacer le dossier "nom" dans le répertoire Documents
  • cp nom1 nom2:duplique le dossier nom1, mais le renomme en nom2
  • mkdir -p 1/2: création d'un dossier dans un sous dossier

    • ouvrir avec nano éditeur de texte terminal + "shift + inser" pour coller le texte

Suppression de document

  • rm nom:supprime le fichier "nom" du dossier courant
  • rmdir nom: supprime le dossier du dossier courant

nano man -mia: indentation du fichier
