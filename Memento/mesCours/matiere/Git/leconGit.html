Les commits

Retour

  • git init::
    Initialisation d'un dossier Git
  • git status::
    état du dossier git(voir les fichiers suivis et ceux qui ne le sont pas)
  • git add::
    ajouter des fichiers à suivre (s'ils ne sont pas suivis ou mis à jour)
  • git commit m- "commentaire":
    sauvegarde des fichiers suivis et mettre à jour le commentaire
  • git log:
    sert à voir la dercription fichier Git
  • git remote add origin + urlUserGithub:
    Créer un lien entre le dossier git créé en local vers le destinataire du site Github
  • git push -u origin master:
    Envoyer le travail au site GitHub
  • git pull:
    récupérer un travail sur le site GitHub

Les branches

  • git branch nouvelleBranche::
    Création de nouvelle branche
  • git checkout nomDeLaBranche:
    Changement de branche (on passe de la branche master à une branche nouvelle ou déjà créée)
  • git checkout master:
    Repositionnement vers la branche d'origine (master)
  • git merge brancheNonMaster:
    fusioner une branche secondaire avec la branche master
  • git branch -d nombranche:
    suppression d'une branche
  • git remote set-url origin:
    En cas d'erreur d'adresse de répertoire github
