Commencer une page html

Retour

  1. Indiquer le langage de programmation avec une balise -!DOCTYPE html-, puis une la balise de langage -html-

  2. Il faut retenir que dans la balise de début (-HTML-), une hiérarchie est à respecter.
    Il y aura en premier la balise -head-, dans lequel nous mettons le meta charset, le titre puis le style ou le lien vers la feuille de style

    • Indiquer la langue qui sera utilisé. Pour le texte Français, on utilisera "utf-8"
      Cette balise s'écrira dans un meta charset, dans le cadre head. Ex:
      -meta charset="utf8"-

    • Ensuite, toujours dans la balise head, donner un titre à la page qui sera indiquée sur l'onglet du navigateur, dans la balise -title-

    • Enfin, nous pouvons soit mettre une -style- (-/style- : il faut penser à la refermer)
      Si nous souhaitons copier le style dans une feuille externe au fichier, il faut créer un lien en tapant ceci: -link rel="stylesheet" href="style.css"-
  3. Enfin dans un second temps, nous avons la balise -body-, on l'on écrira tout le contenu de notre page web. Le document HTML est un langage de programmation dont le but est uniquement dans la descrition, il ne se suffit pas à lui même pour mettre en page ou se styliser
